# Raindance map data

This directory holds **derived data** about the canonical Raindance map:

- `heightmap.png` — community-extracted 257×257 16-bit PNG (MIT, sourced from `levizoesch/tribes.mapping`).
- `heightmap.bin` — same data as `heightmap.png` reduced to a single-channel `uint16` blob.
- `canonical.json` — every gameplay object (flag stands, turrets, sensors, generator, inv stations, vehicle pads, command stations, drop points, repair packs) parsed straight from `Raindance.MIS` with exact float coordinates, plus canonical lighting, fog, rain and sun parameters.
- `build_heightmap.py` — converts `heightmap.png` into the C array used by `program/code/raindance_heightmap.h`.
- `HEIGHTMAP_LICENSE` — MIT license for the heightmap PNG.

## Where are the original Tribes 1 assets?

The original Sierra/Dynamix Starsiege: Tribes v1.11 install (the `.MIS`, `.ted`, `.vol` archives, `.cs` configs, and so on — 139 MB) is **not** kept in `master` to keep clones fast. It lives on the `assets-large` branch.

To fetch it locally for tooling work (writing parsers, regenerating headers, extracting models):

```bash
# from the repo root
make assets
```

or manually:

```bash
git fetch origin assets-large:assets-large
git checkout assets-large -- assets/tribes_original
```

This pulls the `assets/tribes_original/` directory from the `assets-large` branch into your working tree without switching branches. The directory is `.gitignore`d in `master` so subsequent commits won't re-add it.

To clean up afterwards:

```bash
make clean-assets
# or manually
rm -rf assets/tribes_original
```

## Where the canonical data comes from

`canonical.json` was generated by `tools/parse_mis.py` from `assets/tribes_original/base/missions/Raindance.MIS`. To regenerate after a parser update:

```bash
python3 tools/parse_mis.py \
    assets/tribes_original/base/missions/Raindance.MIS \
    assets/maps/raindance/canonical.json
```

The parser is general-purpose and works on any T1 `.MIS`. To bulk-extract every map:

```bash
for mis in assets/tribes_original/base/missions/*.MIS; do
    name=$(basename "$mis" .MIS | tr '[:upper:]' '[:lower:]')
    mkdir -p "assets/maps/$name"
    python3 tools/parse_mis.py "$mis" "assets/maps/$name/canonical.json"
done
```
