⌨️ Raindance Asset Editor

Place, rotate, and scale 3D assets on the Raindance map.

  • W Translate mode
  • E Rotate mode
  • R Scale mode
  • Q Toggle world / local gizmo
  • G Toggle snap-to-grid
  • Ctrl+D Duplicate selected
  • Delete / Backspace Remove selected
  • F Focus camera on selected
  • Ctrl+S Export JSON
  • Ctrl+O Import JSON
  • H Toggle this help
  • Left-click on viewport to select / deselect
  • Orbit: Left-drag (off gizmo) · Pan: Right-drag / Middle-drag · Zoom: Scroll

Exported JSON uses Tribes coordinate system (Z-up). Load GLB/GLTF models via the toolbar.
